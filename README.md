# CodeAlpha Image Gallery

A responsive and interactive Image Gallery project developed using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## Features

* Responsive image gallery layout
* Hover animation effects
* Lightbox image preview
* Modern UI design
* Mobile-friendly interface
* Smooth transitions

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Objective

The objective of this project is to create a visually appealing and user-friendly image gallery while improving frontend development skills such as responsive web design, DOM manipulation, and event handling.

## Files Included

* `index.html`
* `style.css`
* `script.js`
